NRW VNB Atlas

Eine einfache Web-App zur Visualisierung von Verteilnetzbetreibern in Nordrhein-Westfalen auf einer OpenStreetMap-basierten Karte.

Aktueller Stand: Mock-Daten / nicht amtlich. Die Flaechen zeigen keine echten Netzgebietsgrenzen oder Zustaendigkeiten.

Zur Kartenansicht

Ziel der Anwendung

Das MVP bereitet die technische Struktur fuer echte GIS-Daten vor: REST-Endpunkte, GeoJSON-Ausgabe, Suche, Filter und eine zugängliche Ergebnisliste neben der Karte.

Mock-Daten fuer Entwicklungszwecke. Keine amtliche Auskunft.
